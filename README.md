# ☕ Nescafé Landing Page Clone

A modern and responsive **Nescafé-inspired landing page** built using HTML and CSS.
This project focuses on UI design, layout structuring, and responsive web development.

---

## 🚀 Features

* 🎯 Clean and modern landing page design
* 📱 Responsive layout (with separate responsive CSS)
* 🎨 Custom fonts and typography (Titillium Web, Poppins, Montserrat)
* 🧭 Navigation bar with links and CTA button
* 🖼️ Multiple sections:

  * Home (Hero Section)
  * Story & Recipes
  * Products Showcase
  * Mission Section
  * Contact Section
* ✨ Smooth UI styling with gradients, images, and animations
* 🔗 Social media icons using Font Awesome

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **Font Awesome** – Icons
* **Google Fonts** – Typography
